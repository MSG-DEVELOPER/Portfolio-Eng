Stack

My main tech stack reflects my preferences and experience, while I always keep an open mind to exploring other technologies that bring value and efficiency.

HTML & CSS

I build responsive and accessible web pages using HTML and CSS, following best practices for structure and design. I also optimize visibility through SEO and ensure inclusive user interfaces

JavaScript & TypeScript

I develop scalable web applications using JavaScript and TypeScript. Proficient in ES6+, I write clean, efficient, and maintainable code, following best practices that promote collaboration and long-term scalability.

Styling & UI Frameworks

Experienced with design frameworks such as Bootstrap and Tailwind, creating responsive and consistent interfaces. I enjoy working with Sass though I recognize it has become less common in today’s market.

Jest

Firebase

DevOps

Theory and practice in DevOps and CI/CD, currently working with GitHub Actions, Docker, deployments on Vercel, and AWS. Limited experience so far but applied practically, with a strong interest in deepening knowledge in this essential area for modern developers.
