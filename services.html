Stack

Mi stack principal refleja mis preferencias y experiencia, aunque mantengo la mente abierta a explorar otras tecnologías que aporten valor y eficiencia.

HTML & CSS

Desarrollo páginas web con HTML y CSS, asegurando diseño responsivo y buenas prácticas. Optimizo visibilidad con SEO y priorizo la accesibilidad para interfaces inclusivas.

JavaScript & TypeScript

Desarrollo aplicaciones web con JavaScript y TypeScript para proyectos escalables. Familiarizado con ES6, escribo código limpio y eficiente, siguiendo buenas prácticas para facilitar mantenimiento y colaboración.

Bootstrap, Tailwind & Sass

Experiencia en frameworks de diseño como Bootstrap y Tailwind, creando interfaces responsivas y consistentes. Disfruto Sass , pero creo que el mercado actual lo ha desplazado un poco.

React & Ecosistema

React es mi tecnología favorita para desarrollar aplicaciones SPA modernas y eficientes. Conozco librerías del ecosistema y puedo integrar Redux para gestión avanzada del estado. Busco seguir creciendo y consolidarme como referente en React.

Next.js

Conocimientos básicos de Next.js para desarrollar aplicaciones escalables con renderizado híbrido y rutas dinámicas, integrando React con funcionalidades de servidor. Motivado para seguir profundizando y aportar valor a proyectos modernos.

Jest

Conocimientos básicos de Jest para tests unitarios y de integración en proyectos React, asegurando la calidad y confiabilidad del código.

Firebase

Experiencia integrando Firebase, especialmente en auth y storage, aplicando soluciones prácticas para crear aplicaciones funcionales y seguras, con interés en profundizar en la plataforma.

DevOps

Teoria y práctica en DevOps y CI/CD, de momento con GitHub Actions, Docker y despliegues en Vercel. Conocimiento limitado pero aplicado de manera práctica, con interés en profundizar en esta área esencial para desarrolladores modernos.
